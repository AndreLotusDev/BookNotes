# Build continuo

Junto com essa materia, é importante ter o build continuo em dia.

Geralmente deve ser executado quando um novo código é incorporado ao codigo fonte remoto.

Quando o build é disparado, ele compila o codigo, roda a suite de testes e dispara um email/notificacao para todos interessados sobre o projeto, principalmente os programadores.
