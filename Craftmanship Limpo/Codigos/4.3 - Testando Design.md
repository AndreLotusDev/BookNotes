## Regra do domínio

`CalculadoraDeDesconto.Calcular(total)`

- `< 100` → desconto **0**
    
- `100 até 199,99` → **10%**
    
- `>= 200` → **15%**
    

---

## Caso mais simples possível

Começamos com o menor cenário possível.

```csharp
using Xunit;

public class CalculadoraDeDescontoTests
{
    [Fact]
    public void Deve_retornar_zero_quando_total_for_zero()
    {
        var calculadora = new CalculadoraDeDesconto();

        var desconto = calculadora.Calcular(0m);

        Assert.Equal(0m, desconto);
    }
}
```

### Implementação mínima

```csharp
public class CalculadoraDeDesconto
{
    public decimal Calcular(decimal total)
    {
        return 0m;
    }
}
```

Aqui usamos a transformação mais simples: retorno constante.

## Segundo teste — primeiro desvio

Introduzimos um novo comportamento.

```csharp
[Fact]
public void Deve_retornar_10_reais_de_desconto_quando_total_for_100()
{
    var calculadora = new CalculadoraDeDesconto();

    var desconto = calculadora.Calcular(100m);

    Assert.Equal(10m, desconto);
}
```

### Nova implementação

```csharp
public decimal Calcular(decimal total)
{
    if (total >= 100m)
        return 10m;

    return 0m;
}
```

Ainda sem generalizar — só o mínimo necessário.

## Terceiro teste — generalização

Agora forçamos a sair do valor fixo.

```csharp
[Fact]
public void Deve_retornar_15_reais_de_desconto_quando_total_for_150()
{
    var calculadora = new CalculadoraDeDesconto();

    var desconto = calculadora.Calcular(150m);

    Assert.Equal(15m, desconto);
}
```

### Nova implementação

```csharp
public decimal Calcular(decimal total)
{
    if (total >= 100m)
        return total * 0.10m;

    return 0m;
}
```

Aqui acontece a primeira generalização real.

## Testes por faixa (mais robustos)

```csharp
[Fact]
public void Deve_retornar_zero_para_valores_abaixo_de_100()
{
    var calculadora = new CalculadoraDeDesconto();

    for (decimal total = 0m; total < 100m; total++)
    {
        Assert.Equal(0m, calculadora.Calcular(total));
    }
}

[Fact]
public void Deve_aplicar_10_porcento_entre_100_e_199()
{
    var calculadora = new CalculadoraDeDesconto();

    for (decimal total = 100m; total < 200m; total++)
    {
        Assert.Equal(total * 0.10m, calculadora.Calcular(total));
    }
}

[Fact]
public void Deve_aplicar_15_porcento_para_200_ou_mais()
{
    var calculadora = new CalculadoraDeDesconto();

    for (decimal total = 200m; total <= 500m; total++)
    {
        Assert.Equal(total * 0.15m, calculadora.Calcular(total));
    }
}
```

## Código final

```csharp
public class CalculadoraDeDesconto
{
    public decimal Calcular(decimal total)
    {
        if (total >= 200m)
            return total * 0.15m;

        if (total >= 100m)
            return total * 0.10m;

        return 0m;
    }
}
```
